Working with API

JSON Placeholder

JSONPlaceholder is a free online REST API that you can use whenever you need some fake data. It can be in a README on GitHub, in code examples on Stack Overflow, or simply to test things locally.

Check it out!

Random User

A free, open-source API for generating random user data. Like Lorem Ipsum, but for people. In August 2013, they set out with a goal to create a free and easy to use service to generate random user data for application testing.

Check it out!

The Sports DB

An open, crowd-sourced database of sports artwork and metadata with a free API. It was built in 2015 to serve a need for a common interface of sports data for various Home Theatre Applications.

Check it out!
